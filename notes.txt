Step-by-Step Guide to Add a New Joker to Your Game
This guide explains how to add a new joker named “Lucky Seven” to your poker game.
 The joker will add +15 points to the hand score if the played hand contains a card with rank ‘7’. Follow these steps to implement it.


 
Step 1: Define the Joker in boot_menu.js
Add the new joker to the availableJokers array in boot_menu.js. This array defines all jokers available for purchase in the shop.
Instructions:

Open boot_menu.js.
Locate the availableJokers array.
Add a new object to the array with the following properties:
id: A unique identifier for the joker (used for the effect function name).
name: The display name of the joker.
rarity: The rarity level (e.g., 'common', 'uncommon', 'rare').
description: A brief description of the joker’s effect.
cost: The cost in game currency to purchase the joker.
image: The path to the joker’s image file.
script: The path to the joker’s effect script.



Example Addition:
Add the following to the availableJokers array:
{
    id: 'lucky_seven',
    name: 'Lucky Seven',
    rarity: 'common',
    description: 'Añade +15 puntos si la mano contiene una carta con rango 7.',
    cost: 4,
    image: './images/jokers/lucky_seven.png',
    script: './jokers/lucky_seven.js'
}

The script path (./jokers/lucky_seven.js) will be created in the next step.

Step 2: Create the Joker’s Effect Script
Each joker has a dedicated JavaScript file that defines its effect logic. This script must define a function named <joker_id>_effect that takes cards (the selected hand) and handResult (the analyzed hand result) as parameters and returns an object with points and mult modifications.
Instructions:

Create a new file named lucky_seven.js in the ./jokers/ directory.
Define the lucky_seven_effect function to check if the hand contains a card with rank ‘7’ and return +15 points if true.

Content for lucky_seven.js:

function lucky_seven_effect(cards, handResult) {
    console.log("[lucky_seven.js] Checking for rank 7 in hand:", cards);
    const hasSeven = cards.some(card => card.rank === '7');
    if (hasSeven) {
        console.log("[lucky_seven.js] Rank 7 found, adding +15 points");
        return { points: 15, mult: 0 };
    }
    console.log("[lucky_seven.js] No rank 7 found, no effect applied");
    return { points: 0, mult: 0 };
}
